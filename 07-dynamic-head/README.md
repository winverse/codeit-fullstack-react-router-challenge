# 챌린지: Dynamic Head 응용

## 학습 목표

- `Helmet` 컴포넌트를 사용하여 여러 페이지에 동적 헤드 정보를 적용할 수 있다.
- 페이지의 특성에 맞게 `<title>`, `<meta name="description">` 태그를 설정할 수 있다.
- 소셜 미디어 공유를 위한 오픈그래프(Open Graph) 메타 태그를 추가할 수 있다.
