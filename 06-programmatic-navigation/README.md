# 챌린지: Programmatic Navigation 응용

## 학습 목표

- `useNavigate` 훅을 폼 제출과 같은 다른 상황에 응용할 수 있다.
- `onSubmit` 이벤트 핸들러 내에서 프로그래밍 방식으로 페이지를 이동시킬 수 있다.
- `replace` 옵션을 사용하여 불필요한 히스토리를 남기지 않을 수 있다.
