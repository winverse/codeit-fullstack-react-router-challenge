# 챌린지: Programmatic Navigation 연습

## 학습 목표

- `useNavigate` 훅을 사용할 수 있다
- 버튼 클릭이나 폼 제출 시 페이지를 이동시킬 수 있다
- `replace` 옵션을 사용할 수 있다
